Proyecto: Sitio web responsivo para Grafica Meza
Archivos incluidos:
- index.html (página de inicio)
- gallery.html (página de galería con filtros)
- assets/css/style.css
- assets/js/main.js
- assets/img/*.svg (placeholders)
Instrucciones:
1. Descomprimir grafica_meza_site.zip
2. Abrir index.html en Visual Studio Code o en el navegador.
3. Para producción, integrar imágenes reales en assets/img y conectar el formulario con backend.
